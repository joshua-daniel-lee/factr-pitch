# Tech Icons for Section 7

This directory contains SVG icons for the Technology section hub-and-spoke diagram.

## Required Icons

### 1. fedcm-icon.svg (Cyan)
- **Color:** #06c0d7 (cyan)
- **Concept:** Shield or lock merged with browser window
- **Style:** Minimalist, flat design, 2-3 shapes
- **Size:** 128x128px SVG
- **Used in:** Bottom node (FedCM)

### 2. ai-engine.svg (Orange) ⭐ HERO
- **Color:** #f77024 (orange), #ec4899 (pink accent)
- **Concept:** Neural network, circuit brain, AI symbol
- **Style:** Modern, slightly more detailed than others
- **Size:** 128x128px SVG
- **Used in:** Right node (SmartReader AI)
- **Note:** This is the hero icon - most prominent

### 3. oauth-jwt.svg (Gray)
- **Color:** #6b7280 (gray)
- **Concept:** Key with digital/hexagonal elements
- **Style:** Minimal, geometric, technical
- **Size:** 128x128px SVG
- **Used in:** Top and Left nodes (OAuth/JWT)

## Design Guidelines
- All icons should have transparent backgrounds
- Use flat design principles
- Suitable for display on white cards
- Must work well at smaller sizes (80x80px)
- Consistent geometric style across all three

## Icon Generation Prompts

### FedCM Identity
```
Create a minimalist, modern icon representing browser-based identity authentication. 
Style: Flat design with geometric shapes
Primary color: Cyan (#06c0d7)
Secondary color: White
Concept: A shield or lock symbol merged with a browser window outline
Details: Clean lines, 2-3 simple shapes maximum, no gradients
Size: 128x128px SVG
Format: Transparent background, suitable for white cards
```

### SmartReader AI (HERO)
```
Create a bold, eye-catching icon representing artificial intelligence and machine learning.
Style: Modern, slightly more detailed than others (this is the hero)
Primary color: Orange (#f77024) 
Secondary color: Pink accent (#ec4899) for depth
Concept: Neural network nodes, circuit brain, or abstract AI symbol
Details: Geometric, tech-forward, conveys intelligence and automation
Size: 128x128px SVG
Format: Transparent background, suitable for white cards with orange border
Special: This icon should be the most prominent/detailed
```

### OAuth/JWT Security
```
Create a clean, professional icon representing secure authentication tokens.
Style: Minimal, geometric
Primary color: Gray (#6b7280)
Secondary color: White
Concept: A key with hexagonal/digital elements, or a token/badge symbol
Details: Sharp edges, technical feel, 2-3 simple shapes
Size: 128x128px SVG
Format: Transparent background, suitable for white cards
```
